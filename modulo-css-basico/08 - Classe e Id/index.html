Com tudo, não é interessante fazer o uso de ID caso você queira estilizar diversas tags ou elementos daquela maneira, isto por quê a tag ID é utilizada para somente um caracter. Mas para resolver isso, podemos fazer o uso da tag Class

Para fazer a estilização de Classes em CSS é um pouquinho diferente de ID's, visto que com as classes você não utilizará o sinal de hashtag (#) já que o sinal correto é um simples ponto (.)
