Tags Semânticas

O que são tags semânticas?

As tags Semânticas servem para organizar o seu site e auxiliar os buscadores do seu navegador na hora da busca. Essas tags consistem em pequenas tags que podem ser utilizadas para deixar o site mais dinâmico. alguns exemplos são:

header

Geralmente utilizada em cabeçalhos de sites

nav

Geralmente, utilizada para nevegação do seu site

Main

A tag Main é utilizada para a criação do corpo do seu site

Section

A tag section é geralmente utilizada para criar uma área de destaque no site, é aconselhado fazer uso dessa tag uma unica vez por site.

Footer

A tag Footer é como o Header. essa tag é destinada a criação dos rodapés dos sites ou dos seus conteiners.

Article

A tag Article é utilizada para escrever textos idependentes dentro do meu HTML. Essa tag geralmente é utilizada em blogs, onde os criadores geram textos.

É indicado que cada tag Article tenha uma tag de título dentro dela (h1 ao h6)

Aside

A tag Aside é utilizada como uma espécie de leitura complementar / Conteúdo de suporte ao assunto principal do nosso site. geralmente a gente encontra essas tags direcionando para outros sites que abordam aquele determinado tema.

figcaption

É uma tag de descrição para imagens

Adress

A tag Adress geralmente é utilizada dentro da nossa tag footer ou no header, e ela serve para colocar o seu contato ou seu endereço.
